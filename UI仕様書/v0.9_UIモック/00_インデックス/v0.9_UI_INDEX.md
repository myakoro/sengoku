## 🎨 UI デザイン方針

### カラーパレット（戦国和風）
- **背景**: 和紙風テクスチャ（#F5F1E8）
- **主要テキスト**: 墨色（#2C2C2C）
- **強調**: 朱色（#C8381D）
- **アクセント**: 藍色（#1E3A5F）
- **警告**: 金茶（#D4A017）

### フォント
- **見出し**: ゴシック体（太字）
- **本文**: 明朝体
- **数値**: 等幅フォント

### レイアウト原則
- **階層表示**: 縦スクロール（役職）、横タブ切替（部隊）
- **状態表示**: 色分けバー + 数値併記
- **戦場UI**: ミニマップ風の簡易表示
- **数値**: コンパクトに、重要度で文字サイズ調整
