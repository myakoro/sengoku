## 和風デザイン要素
- **報告書枠**: 二重線（╔╗╚╝）で巻物・文書風
- **区切り線**: 横線（───）で罫線風
- **進行ボタン**: 札風の角丸四角
- **配色指定**:
  - 収入: 紅色（金銭）
  - 評価: 藍色（官位）
  - 確認ボタン: 墨色
  - 背景: 生成り色
