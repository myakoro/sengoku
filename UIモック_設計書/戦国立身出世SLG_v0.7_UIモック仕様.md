## 画面1：村/町ステータス画面

### 画面の目的
- 村/町の現在の状態を一覧表示
- 人口・治安・経済・産業・名産・道路・商人の情報を確認
- 村の場合は町昇格条件の進捗を確認

### セクション構造
1. **ヘッダ**：村名・種別（村/町）
2. **左サイドバー**：基本情報（人口・治安・経済）
3. **中央エリア**：産業スロット一覧（1〜3スロット）
4. **右サイドバー**：名産・道路・商人情報
5. **フッタ（村のみ）**：町昇格条件の進捗

### 配置説明（ワイヤーフレーム）

```
┌─────────────────────────────────────────────┐
│ 【村/町ステータス】 村名：○○村（村/町）      │
├──────────┬────────────────┬──────────┤
│ 左サイド   │ 中央エリア         │ 右サイド   │
│            │                    │            │
│ ■人口・治安│ ■産業スロット     │ ■名産情報 │
│ 人口：500  │ ┌────────┐     │ 進捗：80%  │
│ 治安：█75% │ │スロット1   │     │ 名産：なし │
│            │ │農業 Lv.3   │     │            │
│ ■経済      │ │進捗：█50%  │     │ ■道路     │
│ 税収：100貫│ └────────┘     │ レベル：道 │
│ 蓄積金：500│ ┌────────┐     │            │
│            │ │スロット2   │     │ ■商人     │
│            │ │鍛冶 Lv.2   │     │ 行商3名    │
│            │ │進捗：█30%  │     │ ・田中商店 │
│            │ └────────┘     │  (常連)    │
│            │ ┌────────┐     │ ・山田屋   │
│            │ │スロット3   │     │  (面識)    │
│            │ │未開発      │     │            │
│            │ └────────┘     │            │
└──────────┴────────────────┴──────────┘
│ ■町昇格条件（村のみ表示）                    │
│ 人口：500/1000 █50%  産業：2/3 █67%        │
│ 取引：15/30 █50%     治安：75/80 █94%      │
└─────────────────────────────────────────────┘
```

### 表示する情報要素

#### ヘッダ
- 村名（`Village.Name`）
- 種別（`Village.Type`: Village/Town）

#### 左サイドバー
- **人口・治安**
  - 人口（`Village.Population`）
  - 治安（`Village.Security`: 0〜100）※プログレスバー表示
- **経済**
  - 税収（`Village.TaxIncome`）
  - 蓄積金（`Village.AccumulatedMoney`）

#### 中央エリア
- **産業スロット配列**（`Village.IndustrySlots[]`）
  - 産業種類（`IndustrySlot.Type`: Agriculture/Smithing/Weaving/Brewing/Mining）
  - レベル（`IndustrySlot.Level`: 1〜10）
  - 進捗（`IndustrySlot.Progress`: 0〜100）※プログレスバー表示

#### 右サイドバー
- **名産情報**
  - 名産進捗（`Village.MeisanProgress`: 0〜100）※プログレスバー表示
  - 現在の名産（`Village.CurrentMeisan`: null or Meisan object）
    - 名産名（`Meisan.Name`）
    - 種類（`Meisan.Type`）
- **道路**
  - 道路レベル（`Village.RoadLevel`: Road/Highway）
- **商人**
  - 商人リスト（`Village.Merchants[]`）
    - 商人名（`Merchant.Name`）
    - 信用ランク（`Merchant.TrustRank`: Acquaintance/Regular/VIP）

#### フッタ（村のみ）
- **昇格条件**
  - 累計取引回数（`Village.TotalTradeCount`）
  - 人口達成度（`Population/1000`）※プログレスバー表示
  - 産業達成度（`ActiveSlots/3`）※プログレスバー表示
  - 取引達成度（`TotalTradeCount/30`）※プログレスバー表示
  - 治安達成度（`Security/80`）※プログレスバー表示

### コンポーネント構造
- **パネル**：村情報ヘッダー
- **パネル**：左サイドバー（人口・治安・経済）
- **プログレスバー**：治安バー
- **カードリスト**：産業スロット一覧（3枚）
- **プログレスバー**：各産業の進捗バー
- **パネル**：右サイドバー（名産・道路・商人）
- **プログレスバー**：名産進捗バー
- **リスト**：商人リスト
- **パネル**：町昇格条件（村のみ）
- **プログレスバー**：各条件の達成度バー（4本）

## 画面2：商人取引画面

### 画面の目的
- 選択した商人の詳細情報を表示
- 商人の信用値・ランク・取引履歴を確認
- 商品リストを表示（実際の取引はv0.7では未実装）

### セクション構造
1. **ヘッダ**：商人名・階層（行商/街商人）
2. **上部エリア**：商人情報（信用値・ランク・累計取引）
3. **左エリア**：商品リスト
4. **右エリア**：信用ランク効果・取引履歴

### 配置説明（ワイヤーフレーム）

```
┌─────────────────────────────────────────────┐
│ 【商人取引】 商人名：田中商店（行商）         │
├─────────────────────────────────────────────┤
│ ■商人情報                                    │
│ 信用値：█████████░ 75/100  ランク：常連     │
│ 累計取引額：5000貫  累計取引回数：25回       │
│ 次回来訪日：1560年5月15日（行商のみ）        │
│ または「毎日取引可能」（常駐商人のみ）       │
├──────────────────┬──────────────────┤
│ ■商品リスト          │ ■信用ランク効果     │
│ ┌────────────┐   │ 手数料率：5%         │
│ │商品名  │価格│在庫│   │ 卸売：可能           │
│ ├────────────┤   │ 高級商品：可能       │
│ │米      │10  │100│   │ 支払猶予：可能       │
│ │鉄      │50  │50 │   │                      │
│ │刀      │200 │10 │   │ ■取引履歴           │
│ │絹      │300 │5  │   │ 1560/4/10  500貫    │
│ └────────────┘   │ 1560/3/25  300貫    │
│                      │ 1560/3/10  200貫    │
└──────────────────┴──────────────────┘
│ [閉じる]                                      │
└─────────────────────────────────────────────┘
```

### 表示する情報要素

#### ヘッダ
- 商人名（`Merchant.Name`）
- 商人階層（`Merchant.Tier`: Peddler/StreetMerchant）

#### 上部エリア
- **商人情報**
  - 信用値（`Merchant.TrustValue`: 0〜100）※プログレスバー表示
  - 信用ランク（`Merchant.TrustRank`: Acquaintance/Regular/VIP）
  - 累計取引額（`Merchant.TotalTradeAmount`）
  - 累計取引回数（`Merchant.TotalTradeCount`）
  - 次回来訪日（`Merchant.NextVisitDate`: 行商のみ）
  - または「毎日取引可能」（常駐商人のみ）

#### 左エリア
- **商品リスト**（`Merchant.Goods[]`）
  - 商品名（`Good.Name`）
  - カテゴリ（`Good.Category`: Food/Material/Weapon/Luxury）
  - 基礎価格（`Good.BasePrice`）
  - 在庫数（`Good.Stock`）

#### 右エリア
- **信用ランク効果**（参照情報）
  - 手数料率（`TrustRank.CommissionRate`）
  - 卸売可否（`TrustRank.WholesaleAvailable`）
  - 高級商品可否（`TrustRank.LuxuryAvailable`）
  - 支払猶予可否（`TrustRank.PaymentDeferralAvailable`）
- **取引履歴**（`Merchant.TradeHistory[]`）
  - 取引日（`TradeRecord.Date`）
  - 取引額（`TradeRecord.Amount`）

### コンポーネント構造
- **パネル**：商人情報ヘッダー
- **プログレスバー**：信用値バー
- **テキスト**：累計取引情報
- **テキスト**：次回来訪日 or 毎日取引可能
- **テーブル**：商品リスト（4列：商品名/価格/在庫/カテゴリ）
- **パネル**：信用ランク効果
- **リスト**：取引履歴（最新5件程度）
- **ボタン**：閉じる

## 画面3：産業・名産開発画面

### 画面の目的
- 産業スロットの状態を一覧表示
- 各産業のレベル・進捗を確認
- 名産の進捗・効果を確認
- 開発アクションの参照情報を表示

### セクション構造
1. **ヘッダ**：画面タイトル
2. **上部エリア**：産業スロット一覧（1〜3スロット）
3. **下部エリア**：名産情報・効果

┌─────────────────────────────────────────────┐
│ 【産業・名産開発】                            │
├─────────────────────────────────────────────┤

│ │スロット1   │スロット2   │スロット3   │     │
│ │農業 Lv.3   │鍛冶 Lv.2   │未開発      │     │

### 表示する情報要素

#### 上部エリア
- **産業スロット配列**（`Village.IndustrySlots[]`）
  - スロット番号（1〜3）
  - 産業種類（`IndustrySlot.Type`: Agriculture/Smithing/Weaving/Brewing/Mining）
  - レベル（`IndustrySlot.Level`: 1〜10）
  - 進捗（`IndustrySlot.Progress`: 0〜100）※プログレスバー表示
  - 次レベルまでの必要進捗（固定値100pt）

#### 下部エリア
- **名産情報**
  - 名産進捗（`Village.MeisanProgress`: 0〜100）※プログレスバー表示
  - 現在の名産（`Village.CurrentMeisan`: null or Meisan object）
    - 名産名（`Meisan.Name`）
    - 種類（`Meisan.Type`）
- **名産候補発生条件**（参照情報）
  - いずれかの産業がLv.5以上で名産候補が発生
- **名産効果**（名産がある場合のみ表示）
  - 信用ボーナス（`Meisan.TrustBonus`: +10〜+30）
  - 価格倍率（`Meisan.PriceMultiplier`: 1.5〜3.0）
  - 商人来訪率UP（`Meisan.MerchantVisitRateBonus`: +20%）
- **開発アクション**（参照情報）
  - 私事「産業開発」で進捗+10〜30pt
  - 私事「名産開発」で名産進捗+10〜30pt

### コンポーネント構造
- **パネル**：画面ヘッダー
- **カードリスト**：産業スロット一覧（3枚のカード）
- **プログレスバー**：各産業の進捗バー
- **テキスト**：次レベルまでの必要進捗
- **パネル**：名産情報
- **プログレスバー**：名産進捗バー
- **テキスト**：現在の名産（名前・種類）
- **パネル**：名産効果（名産がある場合のみ表示）
- **パネル**：名産候補発生条件（参照情報）
- **パネル**：開発アクション（参照情報）
- **ボタン**：閉じる

## デザイン方針

### 和風テイスト（v0.6準拠）
- **色**：落ち着いた茶色・ベージュ・黒・金色
- **フォント**：明朝体または和風ゴシック
- **枠線**：細い黒線または金色の装飾線
- **背景**：和紙風テクスチャ（推奨）

### 情報設計
- 情報量は多いが、プレイヤーが「ひと目で状態を把握できる」こと
- 数値は大きく表示
- 進捗バーは視覚的に分かりやすく
- 重要な情報は色分け・強調表示

### 戦国時代の雰囲気
- 現代的すぎるデザインは避ける
- アイコンは和風モチーフ（家紋・刀・米俵等）
- ボタンは木製風・和紙風
