## 1. 新規：戦場ログ画面

### 🎯 目的
- 戦闘中の情報を**軽量なテキストログ形式**で表示
- プレイヤーが戦術命令（前進／維持／後退）を選択
- 小隊の行動結果、命令遵守状況、損害を時系列で確認

### 📐 画面構成

```
┌─────────────────────────────────────────────┐
│  【戦場ログ】              天正○年○月○日  │
├─────────────────────────────────────────────┤
│                                             │
│  ┌─────────────┐  ┌───────────────────┐ │
│  │  戦術命令    │  │   戦況ログ          │ │
│  │              │  │                     │ │
│  │ [前進]       │  │ ターン1             │ │
│  │ [維持]       │  │ 主人公小隊：前進    │ │
│  │ [後退]       │  │ 命令遵守 ✓          │ │
│  │              │  │                     │ │
│  │              │  │ 敵小隊A：前進       │ │
│  │              │  │ 接敵発生            │ │
│  │              │  │                     │ │
│  │              │  │ 戦闘結果            │ │
│  │              │  │ 味方損害：3名       │ │
│  │              │  │ 敵損害：5名         │ │
│  │              │  │                     │ │
│  │              │  │ ターン2             │ │
│  │              │  │ ...                 │ │
│  │              │  │                     │ │
│  │              │  │ [スクロール]        │ │
│  └─────────────┘  └───────────────────┘ │
│                                             │
│              [戦闘終了]                     │
└─────────────────────────────────────────────┘
```

### 📝 表示項目（必要最小限）

#### 左側：戦術命令パネル
- **タイトル**：「戦術命令」（筆文字風）
- **3つの命令ボタン**：
  - `[前進]` - 敵に接近する
  - `[維持]` - 現在位置を保持
  - `[後退]` - 距離を取る
- **ボタンスタイル**：
  - 和紙テクスチャの背景
  - 木枠の縁取り
  - 選択時は墨色で強調
  - 縦並び配置（上から前進→維持→後退）

#### 右側：戦況ログパネル
- **ターン番号**：「ターン1」「ターン2」...
- **小隊行動ログ**：
  ```
  主人公小隊：前進
  命令遵守 ✓
  ```
  ```
  家臣A小隊：維持
  命令ズレ（前進）
  ```
- **戦闘結果ログ**：
  ```
  接敵発生
  味方損害：3名
  敵損害：5名
  ```
- **混乱発生ログ**：
  ```
  家臣B小隊：混乱発生
  ```
- **離脱ログ**：
  ```
  敵小隊C：離脱
  ```

### 🎨 配色ルール

| 状態 | 色 | 用途 |
|------|-----|------|
| **命令成功** | 青系（#2d5a8c） | 「命令遵守 ✓」 |
| **命令ズレ／混乱** | 黄系（#d4a017） | 「命令ズレ」「混乱発生」 |
| **損害／死亡** | 赤系（#c73940） | 「味方損害」「死亡」 |
| **通常テキスト** | 墨色（#2b2b2b） | 小隊名、行動名 |
| **背景** | 和紙色（#f8f3e7） | パネル背景 |

### 📏 レイアウト詳細

#### 全体構造
- **画面比率**：左30% / 右70%
- **左パネル（戦術命令）**：
  - 固定幅：300px
  - 縦スクロールなし
  - ボタン間隔：20px
  - パディング：30px
- **右パネル（戦況ログ）**：
  - 可変幅（残り全て）
  - 縦スクロール可能
  - 最新ログが下に追加される形式
  - パディング：20px

#### ログ1行のフォーマット
```
[アイコン] 小隊名：行動内容
          └ 補足情報（命令遵守／ズレ／損害）
```

#### 和風装飾
- **パネル枠**：木目調の縁取り（2px）
- **背景**：和紙テクスチャ
- **区切り線**：墨色の細線（1px）
- **フォント**：ゴシック系（明朝は避ける）

### 🔄 操作フロー
1. プレイヤーが命令ボタンを押す
2. ログに「主人公小隊：[選択した命令]」が即座に追加
3. ターン処理が実行される（Antigravity側）
4. 結果がログに順次追加される
5. 戦闘終了まで繰り返し

### 🎨 配色・スタイル
- v0.55の家臣詳細画面のスタイルを完全踏襲

## 📌 補足事項

### UI設計の原則
1. **軽量性最優先**：装飾は最小限、読みやすさを重視
2. **和風テイスト厳守**：墨・和紙・木枠を基調
3. **情報過多の禁止**：必要な情報のみを表示
4. **視線移動の最小化**：戦闘中は左右分割で完結

### 実装時の注意
- このモックは**見た目の設計のみ**
- ロジック（戦闘処理、昇格判定）はAntigravity担当
- HTML/CSS/JSコードは別途実装フェーズで作成
- 既存UIとの整合性を必ず確認
