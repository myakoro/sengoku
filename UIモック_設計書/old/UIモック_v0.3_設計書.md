# 戦国立身出世SLG 0.3 UIモック（Windows PC版）

## ① 画面一覧（名称＋用途）

### **実装対象画面（3種類）**
1. **メイン（日常行動）画面** - 1日1回の行動選択（公務3種＋私事2種）
2. **村ステータス画面** - 村A・村Bの状態確認（人口・治安・収入・開発度）
3. **月次サマリー画面** - 月末処理結果表示（収入・評価・公務実施確認）

### **将来拡張枠（未実装表示のみ）**
- 家臣（未実装）
- 軍事（未実装）
- 調略（未実装）
- 家・一族（未実装）
- 地方（未実装）

## ② 画面遷移図（テキストベース）

```
[メイン画面] ←→ [村ステータス画面]
     ↓
[月次サマリー画面]
     ↓
[メイン画面]（翌月1日）

※左ナビから各画面へ直接遷移可能
※将来拡張メニューは「未実装」ダイアログ表示
```

## **仕様準拠確認**

### ✅ **仕様書v0.3完全準拠**
- 公務3種（巡察・治安維持・書役仕事）
- 私事2種（開発・取引）
- プレイヤーデータ6項目表示（affiliation/name/rank/money/evaluation/role）
- 村データ4項目表示（population/security/income/development）
- 日次・月次フロー対応

### ✅ **非スコープ機能除外**
- 戦闘・調略・婚姻・家臣団等は「未実装」表示のみ

### ✅ **Windows PC最適化**
- 横長レイアウト（16:9/16:10）
- 左ナビ固定・右ステータス固定
- マウス操作前提のボタン配置
- 最低画面幅1280px以上対応

## **技術仕様**

### **画面解像度**
- 対象解像度: 1280px以上（横長前提）
- レイアウト: 左ナビ(200px) + 中央エリア(可変) + 右ステータス(300px)

### **操作仕様**
- マウス＆キーボード操作
- ウィンドウモード前提
- 1日1回の行動制限UI実装

### **データ表示仕様**
- リアルタイム数値更新
- 行動結果ログ表示
- 月次処理時の自動画面遷移
