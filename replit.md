# 戦国立身出世SLG

## 概要

「戦国立身出世SLG」は、戦国時代を舞台にした戦略シミュレーションゲームです。プレイヤーは下級武士として立身出世を目指します。

## プロジェクト構成

### 元のプロジェクト（C# WPF）
- `SengokuSLG/` - C# WPFデスクトップアプリケーション（Windows専用、Replit環境では動作しません）

### Replit用Webサーバー
- `server.js` - Express.jsを使用したシンプルなWebサーバー（ポート5000で起動）
- `index.html` - UIモック一覧ページ
- `package.json` - Node.js依存関係

### UIモックアップ
- `UIモック_HTML/` - HTMLで作成されたUIモックアップ
  - メイン画面_v0.4.html - 日常拠点画面
  - 公務選択画面_v0.4.html - 公務タスク選択
  - 私事選択画面_v0.4.html - 私事タスク選択
  - 行動結果ダイアログ_v0.4.html - 行動結果表示
  - 月次サマリー画面_v0.4.html - 月末成果報告
  - 和風デザインバージョン各種

### ドキュメント
- `仕様書/` - ゲーム仕様書
- `UIモック_設計書/` - UIデザインガイド
- `和風UIデザイン仕様書_StyleGuide.md` - 和風UIスタイルガイド

## 実行方法

```bash
npm start
```

ブラウザで http://localhost:5000 にアクセスしてUIモックアップを確認できます。
