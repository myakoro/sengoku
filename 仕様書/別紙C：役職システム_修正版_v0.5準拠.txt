# 📘 別紙C：役職システム（修正版 / v0.5 準拠）

---

# ■ C-1. 役職とは（根本定義）

役職＝家中での公務・戦闘の権限を示す階層。
主人公と家臣は、同一の役職階層に属し、役職がそのまま「家中での立場・力量」を示す。

役職は大名家の正式官職ではなく、
「家単位の内部的な役職体系」として扱う。

---

# ■ C-2. 役職階層（家中共通・軍事系）

下から上への順序：

1. **従僕（じゅうぼく）**
　家内従者。公務権限なし。戦場では主人公護衛。

2. **徒士（とじ）**
　正式武士の最下位。公務実行可能。戦場では前衛。

3. **組頭（くみがしら）**
　小隊（徒士）を率いる下級指揮官。治安維持などの現場指揮。

4. **部将（ぶしょう）**
　中規模部隊の指揮官。主人公のv0.5出世上限。

5. **侍大将（さむらいだいしょう）**
　大隊規模の軍団長。v0.7以降の上位役職。

---

# ■ C-3. 役職の意味（ゲーム内での役割）

### ◎ 1. 公務権限
役職が高いほど扱える公務が増える。
- 徒士：基礎公務
- 組頭：中規模公務
- 部将：広域治安・兵站
- 侍大将：軍団運用

### ◎ 2. 戦場での立ち位置
- 従僕：主人公護衛
- 徒士：前衛兵
- 組頭：小隊指揮
- 部将：部隊長
- 侍大将：軍団長

### ◎ 3. 給金（家により定まる）
- 役職で月俸が決まる。
- 従僕のみ家臣が雇用主（主人公）から支給。

---

# ■ C-4. 役職と家格の関係

家格＝家柄・社会的信用  
役職＝公務上の地位  
両者は別軸。

- 家格が低くても、役職が上がれば権限が拡大する。
- 家格が高くても、低役職（徒士以下）に留まることもあり得る。

---

# ■ C-5. 従僕（家内役職）の詳細

### 定義
- 主人公家に属する家内従者（家臣）。
- 年齢・能力は自由。
- 名前管理され、戦闘参加可能。
- 公務不可、徒士昇格で公務可能に。

### 戦闘での扱い
従僕は主人公護衛小隊として参戦する。

小隊戦力：
```

主人公 + (筆頭従僕 ×20%) + (他従僕合計 ×10%)

```

### 従僕枠
主人公の役職で枠が変動：
- 徒士：3
- 組頭：2
- 部将：0
- 侍大将：0

### 昇格
従僕 → 徒士は **主人公が無条件で任命可能**

---

# ■ C-6. 徒士（正式武士）

- 正式な武士階層の最下位。
- 公務実行可能。
- 俸給は主家（羽柴家など）から支給。
- 戦場では前衛兵として参加。

---

# ■ C-7. 組頭（小隊長）

- 徒士数名を率いる小隊指揮官。
- 治安維持・巡察・盗賊討伐などを担う。
- 徒士・従僕へ指示可能。
- 主人公が組頭の場合、従僕枠が2になる。

---

# ■ C-8. 部将（中隊長）

- 戦闘単位の中核指揮官。
- 中規模部隊を率いる。
- 主人公が昇格可能な上限（v0.5）。

※ 部将以上の階層では、従僕を持たない。

---

# ■ C-9. 侍大将（軍団長）

- 家中で最上位の軍事役職。
- 大規模作戦の指揮を行う。
- v0.7以降で解禁。
- 主人公・家臣ともに出世で到達可能。

---

# ■ C-10. 補佐官との関係

補佐官は役職とは別枠。
- 任命可能：従僕 / 徒士 / 組頭 / 部将
- 効果：能力差分40%を主人公の政務系能力に補正
- 個人戦闘能力への補正は一切なし

---

# ■ C-11. 兼任について

同時に複数の役職を持つことは **不可**。  
（v0.5の役職体系は“家内部の単線構造”のため）

※ 「軍事・政務・調略の兼任」の旧仕様は破棄。  
※ 公務の行動は役職ごとに決まる。

---

# ■ C-12. 役職による行動制限

- 従僕：公務不可、護衛のみ  
- 徒士：基礎公務  
- 組頭：中規模公務  
- 部将：広域公務  
- 侍大将：大規模指揮

---

# ■ C-13. 出世の軸

出世の軸は **純粋に役職**。

```

従僕 → 徒士 → 組頭 → 部将 → 侍大将

```

- 主人公も家臣も同じ軸で動く。
- 家臣の昇格は主人公の役職未満まで可能。

---

# ■ C-14. 役職と給金

- 従僕：主人公が支給（月10〜20文）
- 徒士以上：主家（羽柴家など）から支給
- 役職が上がるほど俸給は増加する

（詳細俸給は別紙Iと統合）

---

# ■ C-15. 役職のゲーム内まとめ

```

出世の軸 → 役職
能力補正 → 補佐官
戦闘強さ → 主人公＋従僕（護衛小隊）
経済力 → 私事（知行）
家の格 → 家格システム

```

各システムは明確に分離し、干渉しない設計。

---

# ■ C-16. この別紙での「廃止／変更」項目（明確化）

- 「足軽」役職 → **徒士** に統一（足軽は兵種として残す）
- 「無役」 → **従僕** として正式役職化
- 「軍事/政務/調略の役職分離」 → **家中単線構造に統合**
- 「兼任ペナルティ」 → 廃止（そもそも兼任しないため）
- 「幕僚システム」 → v0.7以降に移動
- 「家老・代官などの政務役職」 → この別紙では扱わない（別紙D/Kに委譲）

---

# ■ C-17. 最終まとめ

この別紙Cは  
**v0.5以降のすべての“役職・出世・家臣階層”の基礎定義**。

- 主人公も家臣も同じ役職軸で扱う  
- 従僕→徒士は主人公裁量で即時  
- それ以上は功績判定  
- 戦闘補正は従僕、政務補正は補佐官  
- 大名家の正式官職とは切り離して扱う

この統一により、  
出世・育成・家臣団が完全に噛み合う。
