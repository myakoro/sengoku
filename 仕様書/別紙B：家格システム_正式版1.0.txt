# 📘 **別紙B：家格システム（正式版 / 1.0）**

---

# ■ B-1. 家格とは何か（根本定義）

**家格＝家としての社会的評価**
であり、

* 出自
* 歴史（代々の積み重ね）
* 信用（貸し借り・人脈）
* 家臣団の質
* 知行地の規模
* 代々の役職歴
* 商人からの信頼

などの「家の格」を総合して示すパラメータ。

### ✔ 個人ではなく「家そのもの」のステータス

### ✔ 世代交代しても基本的に引き継がれる

---

# ■ B-2. 家格の構造（7段階）

家格は7段階制（下ほど強い）。
名称はゲーム内では数値（1～7）で示す。

```
7：名門上級
6：名門中級
5：名門下級
4：中堅家格
3：庶家上級
2：庶家中級
1：庶家下級（初期想定）
```

名称は内部的な概念で、表示は「家格：1～7」。

### ✔ 家格1からでも出世可能

### ✔ 家格7でも弱体化すれば下降あり

---

# ■ B-3. 家格が影響する範囲（極めて重要）

家格は **能力値には一切影響しない。**
影響するのは“社会的扱い”。

### ① 商人（信用レベル）

* 家格1～2 → 村商人レベルが限界
* 家格3～4 → 街商人とも取引可
* 家格5～7 → 地方商人・都市商人と交渉可能
  → 商人信用は家格に強く影響される

### ② 婚姻（縁談）

* 家格60％ × 役職40％ の影響
* 家格が高いと上位家の縁談が通る
* 家格が低くても高役職なら婚姻可能

### ③ 家臣団の質

「役職7：家格3」比率で決定（あなたの指定）

* 役職が高いと高質の家臣が来る
* 家格が高いと“更に優秀な者”が来る
* 家格だけ高くても高質家臣はこない（役職依存が大きい）
  → 家格は“補正”として効く

### ④ 人脈（寺社・商人・豪族など）

家格が高いほど
紹介・斡旋を受けやすい。

### ⑤ 貸し借りの初期値

* 名門はスタート時から一定の信用を持つ
* 庶家はゼロに近いところから積み上げる

---

# ■ B-4. 家格の上昇要因（累積型）

家格は数値的な「ポイント」ではなく、
**実績・人脈・経済力・家の歴史の総合判断** で
段階的に昇格する。

上昇要因：

### ① 長期にわたる役職歴

* 同じ役職を長く務める
* 上位職に任官する
  → 家格の基礎が積み上がる

### ② “家として”の功績

（あなた＋家臣団の働き）

* 公務で大功
* 上司・大名への大きな貸し
* 戦闘で大手柄
* 大規模普請
* 方面軍の副将など重任

### ③ 人脈の広さ

* 寺社からの信用
* 大商人との付き合い
* 他家家臣からの評価
* 大名からの覚え

### ④ 家臣団規模

* 高質家臣が多いと家格が自然上昇
* 多数の家臣を雇える経済基盤がある証拠

### ⑤ 名産品・商業

* 名産の開発
* 大商人との継続的な取引

### ⑥ 世代継承

* 前代の蓄積が次代に引き継がれる
  　（あなたの指示どおり：貸し借りは30%残存）

---

# ■ B-5. 家格の下降要因（現実的）

リアリティを出すため、家格は **上がるだけではなく下がる**。

下降要因：

* 主人（プレイヤー）が失態
* 大きな反乱・謀反
* 大名または上司からの不興
* 連続した戦功ゼロ
* 知行地の大損壊
* 家臣団が壊滅
* 他家への面子を潰す行為
* 世代交代での力不足

家格は「家の信用」なので
信用を毀損すれば下がる。

---

# ■ B-6. 世代交代時の処理

代替わりすると：

### ✔ 引き継ぐもの

* 家格（そのまま）
* 家臣団
* 人脈
* 貸し借り（30%）
* 商人信用
* 名産品・知行地

### ✔ 下がるもの

* 忠誠（家臣の忠誠-20）
* 貸し借り（70%消滅）
* 上司や大名への評価が一時リセット（軽度）

---

# ■ B-7. 家格と出世の関係

### ✔ 出世ロジックは

「功績＞評価＞相性＞貸し借り＞家格」
の順で重みがある。

家格は

* 縁談
* 商人
* 家臣の質
* 人脈

に強く効き、
**出世には“直接のボーナスを与えない”。**

ただし
**家格が高いほうがチャンスには触れやすくなる。**

---

# ■ B-8. 家格のゲーム的意味まとめ

```
出世に必須 → 役職・功績
家の豊かさ → 知行と名産品
家の格調 → 家格
人材の厚さ → 家臣団
金の流れ → 商人信用
婚姻の格 → 家格×役職
```

家格は
**“社会的な格付け”として明確な効果はあるが、
出世を阻害する壁にはしない（あなたの方針どおり）。**
