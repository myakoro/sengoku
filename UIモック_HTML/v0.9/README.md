## 🎨 デザイン方針

### コンパクト版の特徴
- **情報密度を向上**: カラム数を増やして一画面に多くの情報を表示
- **フォントサイズ縮小**: 12px〜13pxをベースに視認性を保ちながら圧縮
- **余白の最適化**: パディング・マージンを調整
- **グリッド拡張**: 最大6カラムまで対応

### 戦国和風デザイン
- 和紙風背景テクスチャ
- 墨色・朱色・藍色の配色
- 状態別の色分けプログレスバー

## 🔧 実装者（Antigravity）向けガイド

### ファイル名について
- **日本語ファイル名を採用**: 仕様書との対応が明確
- **番号付き**: 01〜12の連番で順序が明確
- **実装時の参照**: ファイル名から画面内容が即座に判別可能

### CSS構成
```css
/* 推奨: コンパクト版 */
<link rel="stylesheet" href="css/compact.css">

/* 通常版（参考用） */
<link rel="stylesheet" href="css/common.css">
```

### 主要なCSSクラス

#### レイアウト
- `.grid-2` 〜 `.grid-6`: グリッドレイアウト（2〜6カラム）
- `.card`: カード型コンテナ
- `.compact-card`: コンパクトカード

#### 状態表示
- `.progress-bar`: プログレスバー
- `.progress-fill`: バーの塗りつぶし部分
- `.progress-training-high/mid/low`: 訓練度の色分け
- `.progress-morale-high/mid/low`: 士気の色分け
- `.progress-fatigue-low/mid/high`: 疲労の色分け

#### ボタン
- `.btn-primary`: 主要アクション（朱色）
- `.btn-secondary`: 副次アクション（藍色）
- `.btn-outline`: アウトラインボタン

#### バッジ
- `.badge-primary/secondary/success/warning/danger`

#### アラート
- `.alert-info/warning/danger/success`

## 💡 実装のポイント

### データバインディング
```html
<!-- 例: 訓練度の表示 -->
<div class="progress-bar">
  <div class="progress-fill progress-training-mid" 
       style="width: {{ trainingLevel }}%;">
    {{ trainingLevel }}
  </div>
</div>
```

### 状態による色分け
```javascript
// 訓練度の色分けロジック
function getTrainingClass(value) {
  if (value >= 80) return 'progress-training-high';
  if (value >= 50) return 'progress-training-mid';
  return 'progress-training-low';
}
```

### グリッドの動的生成
```html
<!-- 中隊カードの繰り返し -->
<div class="grid grid-3">
  {% for company in companies %}
  <div class="compact-card">
    <h4>{{ company.name }}</h4>
    <!-- ... -->
  </div>
  {% endfor %}
</div>
```
